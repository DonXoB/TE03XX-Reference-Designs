Simple LED blinkin project for TE0300 boards.
For TE0300 board with xc3s1600e-4fg320 FPGA use blinkin-TE0300-1600 project.
For TE0300 board with xc3s1200e-4fg320 FPGA use blinkin-TE0300-1200 project.
